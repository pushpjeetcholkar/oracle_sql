Chapter 18: Regular Expression Support

Pattern matching with REGEXP functions

1. Regular Expression Functions

Function Purpose
REGEXP_LIKE Pattern matching (similar to LIKE)
REGEXP_SUBSTR Extract substring matching pattern
REGEXP_REPLACE Replace text matching pattern
REGEXP_INSTR Find position of pattern match

2. REGEXP_LIKE Examples

-- Find names starting with vowels SELECT ename FROM emp WHERE REGEXP_LIKE(ename, '^[AEIOU]'); -- Find phone numbers (pattern: ###-###-####) SELECT * FROM customers WHERE REGEXP_LIKE(phone, '^\d{3}-\d{3}-\d{4}$'); -- Case insensitive search SELECT ename FROM emp WHERE REGEXP_LIKE(ename, 'smith', 'i');

3. REGEXP_SUBSTR Examples

-- Extract email domain SELECT email, REGEXP_SUBSTR(email, '@(.+)', 1, 1, NULL, 1) as domain FROM users; -- Extract first word SELECT full_name, REGEXP_SUBSTR(full_name, '\w+') as first_name FROM employees;

4. Common Patterns

-- Email validation WHERE REGEXP_LIKE(email, '^[A-Za-z0-9._%+-]+@[A-Za-z0-9.-]+\.[A-Za-z]{2,}$') -- Phone number formats WHERE REGEXP_LIKE(phone, '^\(\d{3}\) \d{3}-\d{4}$') -- (123) 456-7890 WHERE REGEXP_LIKE(phone, '^\d{3}-\d{3}-\d{4}$') -- 123-456-7890 -- Postal codes WHERE REGEXP_LIKE(zip, '^\d{5}(-\d{4})?$') -- 12345 or 12345-6789
